## Date: 10/11/2025

### Description:

#### PhysiVerse is an interactive platform for virtual physics laboratories. It allows students to explore physics experiments online through pre-added simulations,videos and instructional content.The platform is designed for learning, observation, and enhancing understanding of physics concepts.

### User Stories :

- As a user, I want to view all labs in the system.
- As a user, I want to view experiments inside each lab.
- As a user, I want to see the simulation data, image, or video for each experiment.
- As a user, I want to post comments on experiments.
- As a user, I want to reply to other usersâ€™ comments.
- As an admin, I want to create labs and experiments through the admin panel.
- As a user, I want to log in and log out.

---

### Future Enhancements:

- As a user, I want to edit or delete my comments.
- As an admin, I want to delete or update experiments.
- As a user, I want to search for experiments by name or topic.
- As a user, I want to see when each experiment was added.
- As a user, I want to view and edit my profile information.
- As a user, I want to have a personalized dashboard for saved experiments.
- As a user, I want to share experiment results or notes with others.
- Add AI-powered experiment recommendations.
